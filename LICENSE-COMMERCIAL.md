# LiveCap Core Commercial License Overview

LiveCap Core is dual-licensed. Organizations that cannot comply with the
AGPL-3.0 terms may acquire a commercial license from PineLab.

## Key Points

- **Scope**: Allows integration of LiveCap Core into proprietary products or
  services without the reciprocal obligations of the AGPL.
- **Deliverables**: Access to the same source code as the open-source release,
  with additional support SLAs defined in the contract.
- **Pricing & Terms**: Determined case-by-case based on usage volume, number of
  deployed seats, and support requirements.

## How to Obtain

1. Reach out to PineLab via the following channels:
   - Email: `licensing@pinelab.example.com`
   - Discord: LiveCap Official Server (`#licensing` channel)
2. Provide a brief overview of your intended usage and deployment surface.
3. The PineLab team will respond with contractual terms, pricing, and next
   steps.

For security disclosures or urgent production issues, use
`security@pinelab.example.com` and include "LIVEcap Core" in the subject line.
